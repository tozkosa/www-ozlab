Research 研究テーマ

We study the theory of artificial intelligence and machine learning and their applications.
人工知能・機械学習の基礎とその応用を研究しています。

AI, Machine Learning 人工知能, 機械学習

Information Geometry, statistical Mechanical Informatics, Statistical Learning Theory
情報幾何、情報統計力学、統計的学習理論

Computer Vision コンピュータビジョン

Pattern recognition, object detection, facial expression recognition, generative models, image translation, etc.
パターン認識、物体検出、顔表情認識、画像生成、画像変換など

Audio Signal Processing 音響信号処理

Voice recognition, age recognition, audio generation, anomaly detection, etc.
音声認識、年齢認識、音響生成、異常検知など

Reinforcement Learning 強化学習

Robot control, gliding locomotion, game AI, Sim2Real
ロボット制御、グライド推進運動、ゲームAI、Sim2Real

Natural language Processing 自然言語処理

Chatbot, emotion recognition, natural language understanding
チャットボット、感情分析、自然言語理解

Jewelry Photo Editing 宝飾品画像の編集

Collaboration with Selby Co., Ltd.
株式会社セルビーとの共同研究

Civil Engineering 土木工学

Collaboration with Hosei Univ. Machine learning approach to concrete condition assessment using impact-echo method.
法政大学との共同研究　コンクリート打音探査への機械学習の応用
